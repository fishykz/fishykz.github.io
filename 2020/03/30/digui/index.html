力扣17

思路

  1. 确定递归边界:
    1.输入字符在[0,9]之间且不等于1
    assert(input>=’0’&&input<=’9’&&input!=’1’)
    2.当递归次数=输入字符串长度时单次搜索完毕
    if(index>=digits.size()
    {
    res.push_back(str);
return;
    }
    3.输入空字符串需无返回,而不是返回空字符串
    if(!digits.size())
    return ;
  2. 先构造一个函数findit,读取一位输入的字符串并把它转化为对应字母(所有情况),并拼接在传入的字符串后面
    findit(const string &digits ,int index , const string &s)
    输入字符串     循环读取的字符串位数  前方搜索结果
    1.字符串引用需用const string &
  3. result=findit(digits,0,findit(digits,0,……findit(digits,0,””)))
    将本次调用的结果当参数给下一次调用

递归调用特征:要返回——回溯算法
